== Description ==
Плагин для демонстрации программирования административной части плагинов.
В нем есть только его настройки, образцы полей и больше ничего. 
Скачать можно здесь — https://github.com/vvhromov/vhap

== Changelog ==

Добавлен вывод икноки плагина в Top Level Menu

superhero - dashicons-superhero

vhap_icon - radio buttons - выбор иконки плагина в секции "Настройки плагина"
